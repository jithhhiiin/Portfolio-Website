A personal developer portfolio website built to showcase my projects, skills, and experience.

## Features

- Responsive portfolio design
- About section
- Skills section
- Projects showcase
- Contact section
- Smooth navigation

## Tech Stack

- React.js
- JavaScript
- HTML
- CSS

## Installation

Clone the repository:

git clone https://github.com/jithhhiiin/Portfolio.git

Install dependencies:

npm install

Run the project:

npm start
